## Theme Variables

The heading typography styles are provided by global text utilities:  
`text-heading-4xl`, `text-heading-3xl`, `text-heading-2xl`, `text-heading-xl`, `text-heading-lg`, `text-heading-md`, `text-heading-sm`, `text-heading-xs`. These utilities in turn utilize global type tokens. In addition to the <Link href="/?path=/docs/tokens-typography-tokens--page">ones listed here</Link>, they are:

| Name                            | Description | Default                      |
| ------------------------------- | ----------- | ---------------------------- |
| `--bloom-line-height-heading`   | Line height | `--bloom-line-height-tight`  |
| `--bloom-line-height-content`   | Line height | `--bloom-line-height-normal` |
| `--bloom-type-heading-size-4xl` | Font size   | `--bloom-font-size-4xl`      |
| `--bloom-type-heading-size-3xl` | Font size   | `--bloom-font-size-3xl`      |
| `--bloom-type-heading-size-2xl` | Font size   | `--bloom-font-size-2xl`      |
| `--bloom-type-heading-size-xl`  | Font size   | `--bloom-font-size-xl`       |
| `--bloom-type-heading-size-lg`  | Font size   | `--bloom-font-size-lg`       |
| `--bloom-type-heading-size-md`  | Font size   | `--bloom-font-size-base`     |
| `--bloom-type-heading-size-sm`  | Font size   | `--bloom-font-size-sm`       |
| `--bloom-type-heading-size-xs`  | Font size   | `--bloom-font-size-xs`       |

Note that on a mobile screen size (640px and below), the heading 2xl-4xl utiliies are stepped down to use heading xl-3xl font sizes.
