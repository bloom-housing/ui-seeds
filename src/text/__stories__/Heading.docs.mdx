## Theme Variables

The heading typography styles are provided by global text utilities:  
`text-heading-4xl`, `text-heading-3xl`, `text-heading-2xl`, `text-heading-xl`, `text-heading-lg`, `text-heading-md`, `text-heading-sm`, `text-heading-xs`. These utilities in turn utilize global type tokens. In addition to the <Link href="/?path=/docs/tokens-typography-tokens--docs">ones listed here</Link>, they are:

| Name                            | Description | Default                      |
| ------------------------------- | ----------- | ---------------------------- |
| `--seeds-line-height-heading`   | Line height | `--seeds-line-height-tight`  |
| `--seeds-line-height-content`   | Line height | `--seeds-line-height-normal` |
| `--seeds-type-heading-size-4xl` | Font size   | `--seeds-font-size-4xl`      |
| `--seeds-type-heading-size-3xl` | Font size   | `--seeds-font-size-3xl`      |
| `--seeds-type-heading-size-2xl` | Font size   | `--seeds-font-size-2xl`      |
| `--seeds-type-heading-size-xl`  | Font size   | `--seeds-font-size-xl`       |
| `--seeds-type-heading-size-lg`  | Font size   | `--seeds-font-size-lg`       |
| `--seeds-type-heading-size-md`  | Font size   | `--seeds-font-size-base`     |
| `--seeds-type-heading-size-sm`  | Font size   | `--seeds-font-size-sm`       |
| `--seeds-type-heading-size-xs`  | Font size   | `--seeds-font-size-xs`       |

Note that on a mobile screen size (640px and below), the heading 2xl-4xl utiliies are stepped down to use heading xl-3xl font sizes.
